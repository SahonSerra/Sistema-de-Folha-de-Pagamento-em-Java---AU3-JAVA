Objetivos do projeto:::

O projeto tem como objetivo desenvolver um sistema interativo para cadastrar colaboradores e calcular o salário final conforme o tipo de vínculo. O programa deve utilizar entrada e saída de dados, estruturas de decisão e repetição, menus de seleção e armazenamento de informações em listas dinâmicas (`ArrayList`).

Também devem ser aplicados conceitos de programação como uso de variáveis e constantes, tipos de dados, cálculos matemáticos, estruturas condicionais (`if`, `else if`, `switch`), laços de repetição (`for`, `while`, `do-while`) e organização do código em classes e métodos simples.

Regras do sistema!

Cada colaborador deverá possuir os seguintes dados básicos:

-- Número de registro
-- Nome completo

!! Veja aqui os três tipos de colaboradores:

Funcionário padrão
Recebe apenas o salário base.
Funcionário comissionado
Recebe salário base + comissão.
A comissão é calculada pela seguinte fórmula: Comissão = (vendas * percentual / 100)
Devem ser informados o valor total de vendas no mês e o percentual de comissão.
Funcionário de produção
Recebe salário base + bônus por produtividade.
O bônus é calculado pela seguinte fórmula: Bônus = (valorPorPeça * quantidadeProduzida)
Devem ser informados o valor por peça e a quantidade de peças produzidas no período.

O valor do salário base é R$ 2.000,00 para todos os colaboradores (utilize uma constante para representar esse valor).
